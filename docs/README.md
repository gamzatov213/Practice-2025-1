## Вариативная часть задания

**Разработка консольного приложения «Тетрис» на языке программирования C.**

В рамках вариативной части проектной работы планируется реализовать классическую игру **«Тетрис»** на языке программирования **C**. Программа будет работать в терминале с использованием библиотеки **ncurses** для отображения пользовательского интерфейса.

Проект реализуется с соблюдением принципов **модульного программирования**: логика игры будет отделена от интерфейсной части. В процессе разработки предполагается реализация всех ключевых механик классической игры «Тетрис»: гравитации, управления фигурами, проверки заполненных линий, системы очков и уровней, а также сохранения состояния игры.

Реализация основывается на открытом руководстве разработчика **Стивена Бреннана**, в котором пошагово описан подход к созданию «Тетриса» с нуля.

**Цели проекта:**
- Освоить практическую работу с языком **C**.
- Изучить возможности библиотеки **ncurses** для создания интерфейса в терминале.
- Получить опыт проектирования и реализации игры с нуля в рамках модульной архитектуры.

**Используемые источники:**
- [Stephen Brennan. On Tetris and Reimplementation. 12 June 2015](https://brennan.io/2015/06/12/tetris-reimplementation/)
- [Репозиторий на GitHub](https://github.com/brenns10/tetris)
